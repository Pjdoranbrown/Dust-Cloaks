# Quick Start - 5 Minutes to Playing Your Game

## Option 1: GitHub Pages (Recommended - Works Everywhere)

### Step 1: Create GitHub Account
Go to https://github.com and sign up (free)

### Step 2: Create Repository
1. Click green "New" button
2. Repository name: `dust-cloaks`
3. Click "Create repository"

### Step 3: Upload Files
1. Click "uploading an existing file"
2. Drag ALL files from your `dust-cloaks-refactored` folder
3. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Click "Settings" tab
2. Click "Pages" in sidebar
3. Under "Source" â†’ select "main"
4. Click "Save"

### Step 5: Play!
Your game is now live at:
`https://YOUR_USERNAME.github.io/dust-cloaks/`

**Done! Share this URL with anyone!**

## Option 2: Play Locally (Testing)

### Windows:
```
1. Open Command Prompt in your dust-cloaks-refactored folder
2. Type: python -m http.server 8000
3. Open browser to: http://localhost:8000
```

### Mac/Linux:
```
1. Open Terminal in your dust-cloaks-refactored folder  
2. Type: python3 -m http.server 8000
3. Open browser to: http://localhost:8000
```
